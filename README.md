# Calculadora-Baseada-em-Texto-em-C
Este projeto consiste em uma calculadora simples desenvolvida em C, que permite ao usuário realizar operações matemáticas básicas de forma interativa. O programa apresenta um menu de opções, permitindo que o usuário escolha a operação desejada, insira as opções e obtenha o resultado.

## Instalação e Pré-requisitos

### Pré-requisitos

- Compilador C compatível com o padrão C99 ou superior;
- Terminal ou prompt de comando para executar o programa;
- Sistema operacional Windows, Linux ou macOS.

### Passos para Instalar

1. Baixe ou clone o projeto;
2. Compile o código.

## Usos e Exemplos

Após baixar ou clonar o projeto, siga os passos abaixo para compilar e executar o jogo no terminal.

### Compilar o Código

No Windows, abra o terminal e digite:

```
gcc Calculadora-Baseada-em-Texto-em-C.c -o Calculadora.exe
```

No Linux/macOS:

```
gcc Calculadora-Baseada-em-Texto-em-C.c -o Calculadora
```

### Executando a Calculadora:

No Windows:

```
Calculadora.exe
```

No Linux/macOS:

```
./Calculadora
```

## Como Utilizar

1. O usuário escolhe uma opção de operação (1 a 5) no menu exibido no terminal;
2. Caso escolha uma operação matemática (1 a 4), o usuário insere os valores das váriaveis;
3. O programa realiza o calculo e o resultado aparece no terminal;
4. O usuário escolhe se deseja realizar outra operação ou encerrar o programa.
